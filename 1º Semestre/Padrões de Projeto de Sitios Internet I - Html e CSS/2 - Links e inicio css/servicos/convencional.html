Sistema Off-set / Convencional

O sistema off-set é indicado para médias e grandes tiragens, pois a produtividade/velocidade é alta e o custo de insumos menor, com isso consegue-se um custo unitário muito menor em relação ao sistema digital. Devido a necessidade de preparação de matrizes (chapas) para impressão e perda de folhas para acerto de máquina, não é indicado para pequena tiragens ou itens personalizados.

  • Envelopes
